# ◈ GeoInsight AI

> **Location-based decision support platform** — Coğrafi veri ve makine öğrenmesini birleştiren lokasyon bazlı karar destek demo platformu.

> ⚠️ **Disclaimer / Uyarı:** All scores and metrics in this project are **simulated (mock ML)**. No real-world environmental, seismic, or demographic data is used. This is a demo application only. / Bu projedeki tüm skorlar ve metrikler **simüle edilmiştir (yapay ML)**. Gerçek çevresel, sismik veya demografik veri kullanılmamaktadır. Bu yalnızca bir demo uygulamasıdır.

---

## 🌍 Overview

GeoInsight AI allows users to **click any location on a map** and instantly receive a multi-dimensional analysis report covering:

| Category | Metrics |
|---|---|
| 🌿 Environmental Quality | Air quality (AQI), green cover, water quality, noise pollution |
| ⚠️ Risk Analysis | Flood risk, seismic risk, fire risk, infrastructure reliability |
| 🏙 Livability | Transport access, education quality, healthcare, economic activity |

Results are displayed as score cards (0–100), a radar chart, and an AI-generated Turkish summary with recommendations.

## 🚀 Quick Start (Local)

### 1. Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

API docs → `http://localhost:8000/docs`

### 2. Frontend

```bash
cd frontend
python3 -m http.server 3000
```

Open → `http://localhost:3000`

> Make sure `API_BASE` in `frontend/app.js` points to `http://localhost:8000`.

---

## 📡 API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/analyze` | Analyze a lat/lng coordinate |
| `GET` | `/heatmap-points` | 120 sample heatmap data points |
| `GET` | `/health` | Health check |

### Example Request

```bash
curl -X POST http://localhost:8000/analyze \
  -H "Content-Type: application/json" \
  -d '{"lat": 41.015, "lng": 28.979, "radius_km": 5}'
```

### Example Response (truncated)

```json
{
  "lat": 41.015,
  "lng": 28.979,
  "environmental_score": 63.4,
  "risk_score": 58.1,
  "livability_score": 71.2,
  "overall_score": 64.2,
  "summary": "Bölge orta düzeyde bir profil sergilemektedir...",
  "recommendation": "Riskler değerlendirildikten sonra karar verilmesi önerilir."
}
```

## ☁️ Deploy to Render

1. Fork / clone this repo
2. Go to [render.com](https://render.com) → **New → Blueprint**
3. Connect your GitHub repo — `render.yaml` is auto-detected
4. After deploy, copy your backend URL and update `frontend/app.js`:

```js
const API_BASE = "https://your-backend.onrender.com";
```

5. Commit & push → frontend redeploys automatically

---

## ⚠️ Data Notice

All scores are generated by a **deterministic pseudo-random function** seeded by coordinates. They are designed to look realistic for demo purposes but **do not reflect actual environmental, geological, or socioeconomic conditions** of any real location.

For a production version, replace the mock scoring in `backend/main.py` with real data sources such as:
- [OpenAQ](https://openaq.org/) — Air quality
- [AFAD](https://www.afad.gov.tr/) / USGS — Seismic data
- [Copernicus](https://www.copernicus.eu/) — Land cover / green areas
- [OpenStreetMap](https://www.openstreetmap.org/) — Infrastructure & POIs
